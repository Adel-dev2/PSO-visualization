================================================================================
  COMMON TROUBLESHOOTING
================================================================================

❓ Canvas is blank?
  → Check browser console (F12) for errors
  → Verify .env.local exists and has correct URLs
  → Try demo mode (no backend required)

❓ WebSocket not connecting?
  → Backend must be running on configured port
  → Check firewall rules (8765 may be blocked)
  → Test with demo mode first

❓ File won't upload?
  → File must be .tsp or .txt format
  → Typical file size: <10KB

❓ Seeing "Demo mode" instead of live updates?
  → Normal if backend unavailable
  → Check network tab (F12) to see failed requests
  → Verify NEXT_PUBLIC_PSO_API_URL value
